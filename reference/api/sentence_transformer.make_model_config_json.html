SentenceTransformerModel.make_model_config_json

opensearch_py_ml.sentence_transformer_model.SentenceTransformerModel.make_model_config_json(self, model_name: Optional[str] = None, version_number: int = 1, embedding_dimension: Optional[int] = None, all_config: Optional[str] = None, model_type: Optional[str] = None) → None

parse from config.json file of pre-trained hugging-face model to generate a ml-commons_model_config.json file. If all required fields are given by users, use the given parameters and will skip reading the config.json,

Parameters

model_name: str = None

Optional, The name of the model. If None, default to parse from model id, for example, ‘msmarco-distilbert-base-tas-b’

version_number: int = 1,

Optional, The version number of the model. If None, default to be 1.

embedding_dimension: int = 768,
Optional, the embedding_dimension of the model. If None, parse embedding_dimension from the config file of

pre-trained hugging-face model, if not found, default to be 768.

all_config: dict = None,

Optional, the all_config of the model. If None, parse all contents from the config file of pre-trained hugging-face model.

model_type: str = None,

Optional, the model_type of the model. If None, parse model_type from the config file of pre-trained hugging-face model.

Returns

None
